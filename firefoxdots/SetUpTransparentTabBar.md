# Add Transparent Tab Bar
1. Open Firefox
2. Go to about:config
3. Search for toolkit.legacyUserProfileCustomizations.stylesheets and enable it
4. Go to about:support
5. Open your profile directory (It opens in Kitty for me, if it does this copy the path and paste it into your file manager)
6. Create a directory called "chrome" there
7. Copy userChrome.css to this folder
8. Fully quit Firefox, (on Linux, use `pkill firefox` if it doesn't work, on Windows open Task Manager and quit Firefox there, on macOS go to the Apple Menu, open Force Quit Applications and quit Firefox)
